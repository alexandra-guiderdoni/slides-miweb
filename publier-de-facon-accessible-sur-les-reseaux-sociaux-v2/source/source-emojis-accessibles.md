# Émojis accessibles — réseaux sociaux et accessibilité

Source condensée depuis le contenu fourni dans le chat le 25 juin 2026.

## Intention

Apprendre à communiquer sur les réseaux sociaux de manière accessible, afin de ne pas exclure les personnes en situation de handicap de la communication digitale.

Les réseaux sociaux ne dispensent pas de garantir le droit à l’information. Les limites propres aux plateformes imposent aussi de prévoir d’autres canaux accessibles, notamment un site Internet accessible et, selon les cas, un affichage physique.

## Problème

Les émojis sont très présents dans les publications. Leur usage peut générer de l’engagement, mais il peut aussi brouiller le message pour :

- les personnes aveugles qui ne voient pas les émojis ;
- les personnes malvoyantes qui les perçoivent mal ou pas du tout ;
- les personnes avec des difficultés de compréhension ;
- les personnes daltoniennes quand la couleur porte une partie du sens.

## Principe général

Il ne s’agit pas de bannir les émojis, mais de les utiliser avec modération. Le message doit rester compréhensible sans dépendre de l’apparence visuelle.

## Règle 1 — Comprendre la signification

Chaque émoji possède un équivalent textuel restitué par certaines technologies d’assistance, par exemple les lecteurs d’écran ou les plages braille. Il faut donc vérifier la signification réelle des émojis utilisés avant de publier.

## Règle 2 — Éviter la multiplication

Quand plusieurs émojis sont répétés, chacun peut être vocalisé. Une suite décorative peut devenir longue, désagréable ou incompréhensible à l’écoute.

Exemple à éviter : plusieurs gyrophare autour de « Nous avons besoin de vous ».

Version préférable : un seul symbole d’avertissement, suivi du message utile.

## Règle 3 — Ne pas remplacer les mots ou les chiffres

Un émoji ne doit pas remplacer un mot essentiel. Le sens peut être ambigu, mal interprété ou vocalisé autrement que prévu.

Un émoji ne doit pas remplacer un chiffre. Deux émojis « 4 » ne seront pas lus comme « 44 ». Pour une statistique comme « 44 % », les chiffres doivent rester du texte.

## Règle 4 — Placer les émojis après l’information

Un émoji placé au début ou au milieu d’une phrase peut retarder l’accès au message ou ajouter des mots parasites dans la vocalisation.

Quand un émoji est utile, le placer plutôt en fin de phrase permet de faire entendre l’information avant l’ornement.

## Conclusion opérationnelle

Quelques règles simples améliorent la portée des publications :

- écrire une charte de publication pour les personnes qui produisent les contenus ;
- vérifier et corriger les publications récentes ;
- garder le message en texte réel ;
- limiter le nombre d’émojis ;
- choisir leur position avec soin ;
- publier aussi sur des canaux accessibles quand l’information est importante.
